## Use case

Use this prompt to generate a reusable operational context document AI tools can reference across support, onboarding, product marketing, UX, documentation, and internal workflows.

Instead of rewriting company information into every prompt, this creates a centralized source of truth for your app, customers, workflows, positioning, and brand voice.

## When to use

Use this when you want to:

- improve AI output quality and consistency
- create a reusable company context file for AI tools
- reduce repetitive prompting
- onboard AI tools to your product and workflows
- generate more accurate support, marketing, or UX outputs
- align team-wide AI usage around shared context
- create a reusable system prompt foundation

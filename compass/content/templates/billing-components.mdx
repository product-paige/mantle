Drop-in Polaris components for Mantle + Shopify. Customer, subscription, and plan data flows through `<MantleProvider>` — no extra wiring.

## Horizontal plan cards

Side-by-side plans for pricing pages and in-app upgrade surfaces. Plan name, price, features, CTA. Stacks on mobile.

Use when merchants need to see every plan at once.

## Highlighted card

A single emphasized plan with an optional badge ("Most popular"). Polaris-tinted background draws the eye.

Use to nudge merchants toward a tier inside an upgrade prompt or conversion flow.

## Vertical cards

Stacked plans for sidebars, modals, and embedded surfaces. Same content shape as horizontal, single column.

Use when the billing surface is narrow.

## How it works

Wrap your app in `<MantleProvider>` with your app ID and the merchant's customer API token. Components read `plans`, `subscription`, `customer` from context. Drop into any Polaris page.
